# Uploads Directory

**IMPORTANT:** This directory contains all uploaded images for the SmartTeam platform.

## Structure
- `/products/` - Product images
- `/banners/` - Banner images
- `/categories/` - Category images
- `/downloads/` - Downloadable files
- `/product-pdfs/` - Product PDF files

## Deployment Notes
- This directory is **excluded from deployment** to preserve uploaded files
- Files in this directory are **never deleted** during publish operations
- Configure publish profiles to skip this directory

## DO NOT DELETE
This directory and its contents must be preserved across deployments.
